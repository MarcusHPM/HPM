
**# HPM - Gas Dilution Calculator**
This calculator calculates the time air/gas needs to flowing reach a target concentration.
This is just an APP version of the Gas-Dilution.xlsx file on the HPM Sever.
Inputs = Cylinder Concentration, Target Concentration, Bag Volume, Flow rate
Outputs = Air time, Gas time
